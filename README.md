# 🚀 Productivity Dashboard

A fully functional **Productivity Dashboard Web App** built using **HTML, CSS, and JavaScript**.  
This project helps users manage tasks, plan their day, track goals, stay motivated, and improve focus — all in one place.

## ✨ Features

### 📝 Todo List
- Add tasks with title & description
- Mark tasks as completed
- Highlight important tasks
- Data persistence using **localStorage**

---

### 📅 Daily Planner
- Hourly planner (6 AM – 12 AM)
- Save daily schedule
- Auto-save functionality

---

### 💬 Motivation Section
- Displays **Quote of the Day**
- Fetches quotes from API dynamically

---

### ⏱️ Pomodoro Timer
- 25-minute work session
- 5-minute break session
- Start / Pause / Reset controls
- Session switching (Work ↔ Break)

---

### 🎯 Goals Tracker
- Add goals with deadlines
- Track status:
  - ✔ Done  
  - ⏳ Active  
  - ❌ Expired  
- Delete goals anytime

---

### 🌤️ Weather Dashboard
- Real-time weather using API
- Displays:
  - Temperature
  - Weather condition
  - Humidity
  - Wind speed
  - Precipitation
- Uses **Geolocation API** for current city

---

### 🕒 Live Date & Time
- Displays:
  - Current day
  - Time (AM/PM)
  - Full date

---

### 🎨 Theme Toggle
- Switch between:
  - 🌙 Dark Mode  
  - ☀️ Light Mode  

---

### 🧭 Navigation System
- Interactive feature cards
- Full-screen sections for each module
- Back button navigation

---
